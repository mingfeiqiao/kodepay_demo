# 项目名称

KodePay Demo

[中文文档](https://juejin.cn/spost/7296837697411645491)


##  环境

-   node>=18.16.0
-   npm>=9.5.1
-   yarn>=1.22.19

##  安装

**1.下载项目** 

```
git clone https://github.com/mingfeiqiao/kodepay_demo.git
```

**2.安装依赖**

```
yarn install
```

##  使用方法

1.运行程序：生成插件解压包

```
npm run build
```

2.加载已解压的插件程序：导入KodePay Demo插件包
